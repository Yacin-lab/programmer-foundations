	File : reverse_params.c
	Allowed functions: write
	----------
	Write a program that takes strings as arguments and displays them in reverse order, one per line.
	Each argument must be printed exactly as received, followed by a newline.

	If there are more than 3 arguments, just print a new line.
	If there are no arguments, the program simply displays new line.

	----------
	 Example:
	$> ./reverse_params one two three | cat -e
	three$
	two$
	one$
	$> ./reverse_params test | cat -e
	test$
	$> ./reverse_params one two three four | cat -e
	$
	$> ./reverse_params | cat -e
	$
